# Project Matts quiz
** Description- This is a python based quiz game thats has a shop that can get items that affect gameplay. It also has a random events that can occer with the shop and questions. This quiz game at its base is just a mulitple choice quiz with extra features sprinkled in.

# Overview
** 11 question Quiz

Has points and coins 

Has a small shop

Has events that depend on the random import

# Gameplay
** Multiple-choice quiz questions

Answer correct questions to get points

Correct answers also give coins

Coins can be used in the shop


Random events can occor


# Shop options 
Extra points for 5 coins

Skip question for 3 coins

# Events at random

** Taxes take 3 coins

Do over repeats the question

# Code
1. For loop for the active code in the game
2. Made a function for shop
3. Question answers/shop input is made with if/then statements
4. Random Number to determine when events happen
5. Trackers for counts,coins,points,questions at the beginning

# Extra notes

** Game is terminal based

Messy, but not hard to add too
